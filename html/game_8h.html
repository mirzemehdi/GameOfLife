Function to see if cell is Alive or not.

That is how game works with its rules. If an alive cell has a 0 or 1 alive neighbours, it will turn to be dead beacuse of under-population. If an alive cell has 4 or more alive neighbours, it will also turn to be dead because of overcrowding. If an alive cell has 2 or 3 alive neighbours it lives on to the next generation. Any dead cell with exactly three live neighbours becomes alive.
